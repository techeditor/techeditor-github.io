2024/05

CloudFront Extensions解决方案v2.2.0新版本发布

CloudFront Extensions是开箱即用的解决方案，帮助客户高效地运维Amazon CloudFront。

新版本主要包括以下创新和功能优化：

  • CDN预热API全新升级。
    • 支持大批量文件批量预热（如20000个URL）和多达240个Pop点的预热任务下达。
    • 支持按区域、国家、PoP点预热。
  • 新增限速功能。
    • Amazon WAF的速率规则的最低速率限制为100查询/5min。
    • 一些客户的服务URL调用了大量的后端数据库资源和付费的第三方API，因此希望能够使用较低的速率规则来保护某些关键URL。

2024/04

推出面向Claude3迁移的Bedrock Access Gateway和Claude Prompt Generator

Bedrock Access Gateway
(加速向Claude3等基于Bedrock的LLM大模型迁移)

该方案一键部署，基于无服务器架构。用户只需要在其AWS账号中部署该方案 (约3分钟)，然后修改环境变量，将针对GPT的调用发给Bedrock Access Gateway，就可以实现原应用中的模型到Claude家族系列模型的自动切换。该解决方案对用户的代码没有侵入性，便于用户快速体验Claude系列模型 (包括最新的Sonnet和Haiku)，加速业务迁移商机。

Claude Prompt Generator
(面向Claude3的提示词转换、优化和评估工具)

该方案利用LLM，并根据Anthropic prompt的最佳实践，帮助客户快速将面向GPT的提示词转换和改进为适用于Claude3模型的高质量提示词，并提供效果优化功能，进一步优化提示词在客户场景中的效果，从而协助客户快速实现生成式AI应用针对Claude模型的适配。该方案提供一套Python代码和Web应用界面，便于使用者直接操作。

推出基于Claude3和Stable Diffusion插件解决方案的AI生图演示达芬奇 (Da Vinci)

CSDC已推出生成式AI生图演示达芬奇 (Da Vinci)，通过将Stable Diffusion亚马逊云科技插件 (ESD) 解决方案连接到Claude3模型，扩展了Stable Diffusion的应用场景。该演示提供了与DALL·E和Midjourney类似的功能，能让客户使用包括中文在内的自然语言调用Stable Diffusion模型生图。它和背后的方案适合技术能力有限、需要低门槛AI生图的目标客户，包括现有的使用DALL·E和Midjourney的客户。

该套方案的主要优势包括：

  • 支持客户自定义Stable Diffusion模型，置换成适合其业务需要的模型，以及ComfyUI的工作流。DALL·E和Midjourney不具备这种扩展能力。
  • 通过Claude3实现提示词扩写，包括正向提示词和负向提示词，大大降低用户的使用门槛，提高生图体验。

该演示的部署包括两步：部署ESD解决方案、启动示例脚本。请联系Wenmeng Gao，获取部署资源，了解更多详情。

2024/03

Stable Diffusion亚马逊云科技插件v1.5.0新版本发布

新版本主要包括以下创新和功能优化：

  • 升级支持Stable Diffusion web UI v1.8.0
  • 支持换脸插件ReActor for Stable Diffusion
  • 新增API推理调试器功能
  • 支持基于Kohya_ss的LoRa模型训练
  • 优化云上资源管理的用户体验
  • 优化BYOC自定义插件的方式，并支持启动优化
  • 支持实时推理节点的弹性伸缩功能
  • 支持所有实例自定义实例数范围，即便自动伸缩，也可以设置最小实例数，而不是0
  • 优化中间件部署时间缩短67%，降低至4分钟
  • 升级Python版本从3.9至3.10
  • web UI资源管理界面用户体验优化

2024/02

日志通解决方案v2.2.0预览版抢鲜用

新版本主要包括以下创新和功能优化：

  • 支持从Windows实例摄取日志。
  • Light Engine支持更多日志源，包括VPC Flow、RDS、CloudTrail等服务日志，Syslog日志，以及以S3为源的应用日志。
  • 自动生成标签。客户可以在AWS计费和成本管理服务中通过“CLOSolutionCostAnalysis”标签查看解决方案成本。
  • 在创建日志流水线时增加lambda并发配置项，以便在需要时设置lambda的并发数。
  • 支持新区域，包括亚太地区 (海得拉巴、雅加达、墨尔本)、以色列 (特拉维夫)、加拿大 (卡尔加里)、欧洲(西班牙、苏黎世)、中东 (阿联酋)。
  • 增加了新版本发布通知，客户可以在CLO的控制台中收到关于新功能的通知。

Stable Diffusion亚马逊云科技插件解决方案v1.4.0新版本发布

新版本主要包括以下创新和功能优化：

  • 将Amazon SageMaker推理节点的冷启动时间缩短70%，减少到3分钟。
  • 支持通过Amazon SageMaker实时推理节点进行文生图推理。
  • 支持Stable Diffusion web UI v1.7.0。
  • 支持同一账号内在多region部署多套本解决方案。
  • 支持在云上通过ComfyUI进行推理。
  • 优化web UI用户体验，新增用户组、角色组、模型和推理节点列表的多维度过滤筛选，提供便利的增删功能。

有关更多内容，请参考更新说明。

Data Transfer Hub v2.6.0新版本发布

新版本主要包括以下功能改进与优化。有关更多内容，请参考更新说明。

  • 支持使用Amazon S3托管密钥加密的Amazon S3目标存储桶。
  • 提供可选的Amazon S3存储桶来保存前缀列表文件。对于以OpenSource为源桶的情况，前缀列表文件可选择存放在自定义的桶内。
  • Finder的内存容量可增加到385GB和512GB，最大单线程扫描文件数提升100%，从而可支持高达1.28 billion个文件的比较。
  • 新增3个支持区域：亚太地区 (墨尔本)、加拿大 (卡尔加里)、以色列 (特拉维夫)
