利用Bedrock Access Gateway实现GenAI应用中的模型切换
Quickly Access Multiple LLMs with the Bedrock Access Gateway solution

:-:-
系列AI/ML解决方案研发
主讲人Aiden Dai
简介本次分享将详细介绍CSDC开发的面向Bedrock的解决方案Bedrock Access Gateway。该解决方案一键部署，基于无服务器架构。用户只需要在其AWS账号中部署该方案 (约3分钟)，然后修改环境变量，将针对GPT的调用发给Bedrock Access Gateway，就可以实现原应用中的模型到Claude家族系列模型的切换。该解决方案对用户的代码没有侵入性，便于用户快速体验Claude系列模型 (包括最新的Sonnet和Haiku)。
